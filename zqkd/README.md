### 中青旧版本

豌豆荚普通下载

https://www.wandoujia.com/apps/7702132/history

安装好后，抓取CK


### 中青脚本获取cookie（推荐）

> 我们利用账号检查脚本(zqkd_check.js)获取

打开app-我的-右上 设置-账号管理，绑定手机号及设置密码

    export zqkdAccount='手机号#密码'

运行zqkd_check.js脚本，获取ck填入环境变量，多账号用@隔开

    export zqkdCookie='uid=***&zqkey=***&zqkey_id=***'
